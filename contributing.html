    # File bug reports

    Please report any bugs to our issue tracker. If you are reporting a bug, please provide as much information as you can including:

    1. Hypothesis-Bio version
    2. Hypothesis version
    3. Python version
    4. OS and version
    5. How to reproduce the error

    # How to contribute

    Ready to contribute to Hypothesis-Bio? Here's how to set up your local environment for development:

    1. Fork the hypothesis_bio repo on GitHub.

    2. Clone your fork locally:

      $ git clone git@github.com:your_name_here/hypothesis_bio.git

    3. Install your local copy into a virtualenv.

      $ cd hypothesis_bio/
$ virtualenv env
$ source env/bin/activate
(env) $ pip install -e .
(env) $ pip install -r requirements-dev.txt

    4. Create a branch for local development:

      $ git checkout -b name-of-your-bugfix-or-feature

      Now you can make your changes locally.

    5. When you're done making changes, check that your changes pass flake8, mypy, black, and isort.

      $ mypy hypothesis-bio/
$ black hypothesis-bio/
$ isort hypothesis-bio/*
$ flake8 hypothesis-bio/

    6. Commit your changes and push your branch to GitHub::

      $ git add .
$ git commit -m "Your detailed description of your changes."
$ git push origin name-of-your-bugfix-or-feature

    7. Submit a pull request through the GitHub website.
